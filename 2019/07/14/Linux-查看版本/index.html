Linux 查看版本

如何查看Linux版本

查看Linux内核版本

  1. cat /proc/version
  2. uname -a

显示系统名、节点名称、操作系统的发行版号、操作系统版本、运行系统的机器 ID 号

  1. uname -r

显示操作系统的发行版号

查看Linux系统版本

  1. lsb_release -a
  2. cat /etc/issue
